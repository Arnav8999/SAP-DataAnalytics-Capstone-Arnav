## 🚀 Project Overview

This project implements an **end-to-end SAP Data Analytics solution** for a retail company.

It processes **2.5 million+ annual sales transactions** and builds a **real-time analytics dashboard** using:

* SAP S/4HANA (Source)
* SAP BW/4HANA (Data Warehouse)
* SAP Analytics Cloud (SAC)

📄 Full Report: See `SAP_DataAnalytics_Project_ArnavSingh.pdf`

---

## 🏢 Business Problem

RetailEdge India lacked:

* Real-time sales visibility
* Centralized reporting
* KPI-based decision making

Reports were:
❌ Delayed (5–7 days)
❌ Excel-based
❌ Error-prone

---

## 🎯 Objectives

* Build centralized data warehouse (BW/4HANA)
* Design ETL pipelines
* Create SAC Dashboard
* Deliver actionable KPIs

---

## 🏗️ Architecture

```
SAP S/4HANA → BW/4HANA (PSA → aDSO → CompositeProvider) → SAC Dashboard
```

---

## 📦 Tech Stack

* SAP S/4HANA
* SAP BW/4HANA 2.0
* SAP HANA Database
* SAP Analytics Cloud (SAC)
* ABAP (Transformations)

---

## 📊 Key KPIs

* Revenue vs Target
* Sales Growth %
* Monthly Trends
* Top 10 Products
* Regional Sales Heatmap
* Store Performance Ranking
